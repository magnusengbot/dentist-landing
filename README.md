# Bright Smile Dental - Landing Page

A modern, responsive landing page for a dental practice.

## Tech Stack

- **HTML5** - Semantic markup
- **CSS3** - Mobile-first responsive design
- **Vanilla JS** - Minimal interactions
- **Netlify Forms** - Contact form handling

## Project Structure

```
dentist/
├── index.html          # Main landing page
├── css/
│   └── styles.css      # Styles
├── js/
│   └── main.js         # JavaScript interactions
├── images/             # Static assets
├── robots.txt          # SEO
├── sitemap.xml         # SEO
└── README.md           # This file
```

## Sections

1. **Header** - Logo, navigation, CTA
2. **Hero** - Practice name, tagline, book appointment
3. **Services** - 6-8 dental services
4. **About** - Dentist bio, credentials
5. **Trust Signals** - Insurance, testimonials
6. **Contact** - Form, address, hours, map
7. **Footer** - Links, social, copyright

## Development

No build step required. Open `index.html` in a browser.

## Deployment

Deployed via Netlify with automatic HTTPS.

- Lighthouse Performance: ≥ 90
- Lighthouse Accessibility: ≥ 90
